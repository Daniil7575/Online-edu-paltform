# Online-edu-paltform
Склонируйте репозиторий в удобную директорию при помощи команды 
```
git clone https://github.com/Daniil7575/Online-edu-paltform.git
```

Создайте в рабочей директории файл `.env` и добавьте в него следующие строки
```
DJANGO_KEY=
DB_NAME=
POSTGRE_USERNAME=
POSTGRE_PASSWORD=
POSTGRE_HOST=
POSTGRE_PORT=
```
